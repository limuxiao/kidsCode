## 📖 项目简介

**CodeQuest Kids** 是一个面向 6-12 岁儿童的编程启蒙教育平台。通过精心设计的互动游戏，让孩子在探索、创造和解决问题的过程中，自然而然地理解编程的核心概念。

### 🎯 教育理念

- **🎨 游戏化学习**：将抽象的编程概念融入有趣的游戏场景
- **🧠 渐进式难度**：从简单到复杂，循序渐进培养计算思维
- **✨ 即时反馈**：可视化执行过程，让孩子看到代码的实际效果
- **🎪 多样化场景**：不同主题的游戏，训练不同的编程技能
- **🤖 AI 辅助**：智能导师提供个性化指导和答疑

### 🌟 核心价值

| 编程概念 | 对应游戏 | 训练效果 |
|---------|---------|----------|
| **序列执行** | 所有游戏 | 理解程序按顺序执行 |
| **循环重复** | 迷宫探险、音乐创作 | 学会用循环简化代码 |
| **条件判断** | 迷宫探险（敌人检测） | 根据情况做决策 |
| **空间推理** | 迷宫探险 | 坐标系统和方向控制 |
| **模式识别** | 魔法药水店、音乐创作 | 发现规律并运用 |
| **调试能力** | 所有游戏 | 从错误中学习 |
| **创造力** | 魔法画布、城堡建造 | 自由创作和表达 |

## 🎮 游戏模块

### 1️⃣ 迷宫探险 (Maze Game) ⭐⭐⭐⭐⭐

**训练效果：最强** | **适合年龄：7-12岁**

<img src="https://img.shields.io/badge/难度-★★★★☆-orange" alt="难度" />

**核心玩法：**
- 使用编程指令控制角色移动（前进、后退、转向）
- 规划路径避开障碍物和墙壁
- 消灭敌人到达终点收集宝藏

**教育价值：**
- ✅ **空间计算思维**：培养坐标、方向、路径规划能力
- ✅ **序列+循环**：FORWARD(5) 理解参数化重复
- ✅ **条件雏形**："敌人在前方→需要ATTACK"
- ✅ **优化思维**：追求最少指令数完成任务
- ✅ **持续挑战**：动态生成关卡，难度无上限

**难度递进：**
```
第1关：直线前进（5×5网格）
第3关：L型转弯 + 墙壁
第5关：引入敌人，需要攻击指令
第10关：8×8网格，多个敌人，复杂路径
第20关：高级迷宫，需要15+条指令
```

**可用指令：**
- `FORWARD(n)` - 向前走 n 步
- `BACKWARD(n)` - 向后退 n 步
- `TURN_LEFT` - 左转 90°
- `TURN_RIGHT` - 右转 90°
- `ATTACK` - 攻击前方敌人

### 2️⃣ 魔法药水店 (Potion Shop) ⭐⭐⭐☆☆

**训练效果：中等** | **适合年龄：6-10岁**

<img src="https://img.shields.io/badge/难度-★★☆☆☆-green" alt="难度" />

**核心玩法：**
- 根据顾客需求调配魔法药水
- 按正确顺序添加原料（星尘、熔岩、草药等）
- 最后搅拌完成配方

**教育价值：**
- ✅ **序列思维**：理解顺序的重要性
- ✅ **需求分析**：将自然语言转换为指令
- ✅ **调试能力**：失败后找出错误并改正
- ✅ **记忆力**：记住配方模式

**当前限制：**
- ⚠️ 缺乏循环、条件等高级概念
- ⚠️ 最多4条指令，天花板较低
- ⚠️ 主要训练记忆而非创造

> 💡 **改进计划**：查看 [`doc/魔法药水店改版方案.md`](doc/魔法药水店改版方案.md) 了解如何升级为全面的编程训练系统

**可用指令：**
- `ADD_STARDUST` - 加入星尘
- `ADD_FLAME` - 加入熔岩
- `ADD_SLIME` - 加入粘液
- `ADD_HERB` - 加入草药
- `ADD_CRYSTAL` - 加入冰晶
- `STIR` - 搅拌药水

### 3️⃣ 音乐创作 (Rhythm Game) ⭐⭐⭐⭐☆

**训练效果：较强** | **适合年龄：6-10岁**

<img src="https://img.shields.io/badge/难度-★★★☆☆-yellow" alt="难度" />

**核心玩法：**
- 使用音符指令创作旋律
- 学习基础乐理（音高、时长）
- 完成指定的音乐片段

**教育价值：**
- ✅ **序列+时间**：理解顺序和持续时间
- ✅ **模式识别**：音乐的重复结构
- ✅ **艺术+逻辑**：培养跨学科思维

**可用指令：**
- `PLAY_NOTE(音高, 时长)` - 演奏音符
- `REST(时长)` - 休止符

**关卡示例：**
```
第1关：简单音阶 (C4, D4, E4)
第2关：小星星片段 (C4 C4 G4 G4)
第3关：音高跳跃 (C4 → C5)
```

## 🚀 快速开始

### 环境要求

- **Node.js** 18+ (推荐 20.x LTS)
- **npm** 或 **yarn**

### 安装步骤

1️⃣ **克隆项目**
```bash
git clone <repository-url>
cd codequest-kids
```

2️⃣ **安装依赖**
```bash
npm install
```

3️⃣ **配置 API Key**

创建 `.env.local` 文件并添加你的 Gemini API Key：
```env
GEMINI_API_KEY=your_api_key_here
```

> 💡 获取 API Key：访问 [Google AI Studio](https://ai.google.dev/)

4️⃣ **启动开发服务器**
```bash
npm run dev
```

应用将在 `http://localhost:3000` 运行

### 构建生产版本

```bash
npm run build
```

构建产物位于 `dist` 目录

### 预览生产版本

```bash
npm run preview
```

## 🎓 教学建议

### 推荐学习路径

**第1阶段：基础入门（1-2周）**
1. 从魔法药水店开始（简单序列）
2. 理解指令按顺序执行的概念
3. 培养试错和调试的习惯

**第2阶段：空间思维（2-3周）**
1. 进入迷宫探险游戏
2. 学习方向控制和路径规划
3. 理解坐标系统

**第3阶段：复杂逻辑（3-4周）**
1. 迷宫中级关卡（敌人、多次转向）
2. 学习隐式循环（FORWARD(5)）
3. 理解条件判断雏形

**第4阶段：创意探索（持续）**
1. 音乐创作和魔法画布
2. 自由发挥创造力
3. 结合多个游戏综合练习

### 家长/教师指导要点

✅ **鼓励探索**：允许孩子试错，失败是学习的一部分
✅ **不直接给答案**：引导思考而不是告诉解法
✅ **庆祝进步**：每个小成就都值得认可
✅ **适当休息**：避免过度疲劳影响兴趣
✅ **结合现实**：将编程概念与日常生活联系

### AI 导师使用建议

- 🤖 当孩子卡住时，先让他们尝试用自己的话描述问题
- 🤖 AI 会引导思考方向，而不是直接给出完整答案
- 🤖 鼓励孩子与 AI 对话，提升表达和逻辑能力
